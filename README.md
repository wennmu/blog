# 博客

## 安装

```shell
npm install
```

## 运行

```shell
npm run dev
```

## 编译

```shell
npm run build
```

## 提交

```shell
cp -rf docs/.vuepress/dist/* ./
git add .
git commit -m "."

```

## 推送

```shell
git push
```
