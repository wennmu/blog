> 通过修改数组切片的值，来改变数组元素的值

```go
package main

import "fmt"

func main() {
    x := [3]int{1, 2, 3}
    y := x[0:1]

    var _ = func(arr []int) int {
        arr[0] = 7
        fmt.Println(arr)
        return 7
    }(y)

    fmt.Println(x)
}

```

执行结果：

```go
[7]
[7 2 3]
```
